Getting started with development

Getting started with development of SlowCheetah is pretty easy. You will need the following to get started:
  • Visual Studio 2010
  • Visual Studio 2010 SDK

Components of SlowCheetah

SlowCheetah is pretty simple, it mainly consists of the following componetns.
  • Visual Studio plugin for UI support
  • MSBuild .targets file to support transforms during build
  • Unit tests

Here is an overview of how SlowCheetah works.

After installing SlowCheetah when the VS package is loaded (*i.e. when the first project is loaded in the VS instance*)

$ git fetch origin
$ git checkout gh-pages
